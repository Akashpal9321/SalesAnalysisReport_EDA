# SalesAnalysisReport_EDA

*About the Project

This project analyzes an e-commerce sales & inventory dataset collected from Kaggle.
The goal is to identify trends in stock levels, popular colors, demanded sizes, and top-performing designs/SKU codes that can help in business decision making.

*Dataset Information

The dataset includes the following columns:

SKU Code

Design No.

Stock

Category

Size

Color

*Objectives

Identify which colors have the highest total stock

Analyze size-wise inventory levels

Find the top 10 SKU codes

Find the top 10 designs

Discover patterns using a correlation heatmap

*Tech Used

Python

Pandas

Matplotlib

Seaborn

*Key Insights

Certain colors dominate the inventory, indicating customer preference or overstocking

L and M sizes show the highest stock, suggesting slower sales or higher production

A few SKU codes hold significantly more stock than others

Top designs help identify what customers likely prefer
